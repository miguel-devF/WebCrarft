# WebCraft — Site de Criação de Sites

Site institucional desenvolvido para apresentar serviços de criação de sites para empresas de todos os tipos.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript puro

## 📄 Páginas
- `index.html` — Página principal
- `obrigado.html` — Página de confirmação após envio do formulário

## ✨ Funcionalidades
- Design responsivo (mobile e desktop)
- Formulário integrado com WhatsApp
- Botão flutuante do WhatsApp
- Animações e efeitos visuais
- Página de obrigado com redirecionamento automático

## 📦 Como usar
Basta abrir o `index.html` no navegador ou usar a extensão Live Server no VS Code.
